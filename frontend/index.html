주요 기능

mcplink가 제공하는 편리한 기능을 통해 MCP 설치 과정을 간소화하고, 클로드 앱 사용을 더욱 확장해보세요.

mcplink란 무엇인가요?

mcplink는 여러분이 원하는 MCP 콘텐츠를 쉽고 빠르게 검색하고 설치할 수 있도록 돕는 서비스입니다. 복잡한 설치 과정 없이 몇 번의 클릭만으로 클로드 앱에 다양한 MCP를 적용할 수 있습니다.

쉬운 검색

간편한 검색 기능으로 원하는 MCP를 빠르게 찾을 수 있습니다.

자동 설치

클릭 한 번으로 MCP를 다운로드하고 올바른 위치에 자동으로 설치합니다.

안전한 사용

검증된 MCP만 제공하여 클로드 앱을 안전하게 사용할 수 있습니다.

mcplink는 클로드 앱의 기능을 확장하고 사용 경험을 향상시키기 위해 개발되었습니다. 저희는 사용자 여러분이 다양한 MCP를 쉽고 안전하게 이용할 수 있도록 끊임없이 노력하고 있습니다.

시작하기 전

mcplink를 사용하기 전에, 다음 두 가지가 준비되어 있는지 확인해주세요:

  • 클로드 앱 설치: MCP를 적용할 대상인 클로드 앱이 컴퓨터에 설치되어 있어야 합니다.
    클로드 앱 설치하기
  • mcplink 데스크탑 앱 설치: mcplink 데스크탑 앱이 컴퓨터에 설치되어 있어야 합니다. (아직 설치하지 않았다면 아래 튜토리얼의 1단계를 먼저 진행해주세요.)

MCP 설치 튜토리얼

1

mcplink 데스크탑 앱 다운로드 및 설치

mcplink 데스크탑 앱을 다운로드하고 설치하는 방법은 다음과 같습니다.

  1. mcplink 웹사이트 방문: mcplink 웹사이트에 접속하여 다운로드 섹션을 찾습니다.
  2. 다운로드 링크 클릭: mcplink 데스크탑 앱 다운로드 링크를 클릭하여 설치 파일을 다운로드합니다. 운영체제에 맞는 설치 파일을 선택하세요.
  3. 설치 파일 실행: 다운로드 받은 설치 파일(예: .exe, .dmg 등)을 실행합니다.
  4. 설치 지시 따르기: 화면에 나타나는 지시에 따라 앱을 설치합니다. 기본 설정으로 설치하는 것을 권장합니다.
  5. 설치 완료 및 앱 실행: 설치가 완료되면 mcplink 데스크탑 앱을 실행합니다.
  6. 시작하기: 앱이 성공적으로 실행되면, mcplink 데스크탑 앱을 이용할 준비가 완료됩니다.
알림

설치 과정에서 문제가 발생하거나 도움이 필요하다면, mcplink 웹사이트의 FAQ를 참조하거나 팀 SCP-201에 문의해주세요.

2

MCP 검색 및 찾기

mcplink 데스크탑 앱을 실행한 후, 설치하고 싶은 MCP를 검색하고 찾아보세요.

  1. 검색창 찾기: mcplink 데스크탑 앱 메인 화면에서 검색창을 찾습니다. 일반적으로 화면 상단 또는 중앙에 위치합니다.
  2. 검색어 입력: 설치하고 싶은 MCP의 이름, 제작자, 또는 관련 키워드를 검색창에 입력합니다.
  3. 검색 시작: 키워드를 입력한 후 검색 버튼을 클릭하거나 Enter 키를 누릅니다.
  4. 검색 결과 확인: 검색 결과 목록이 화면에 표시됩니다. 원하는 MCP를 찾았는지 확인해보세요.
  5. 상세 정보 확인: 검색 결과 목록에서 MCP 항목을 클릭하면 해당 MCP에 대한 상세 정보 페이지로 이동합니다. 상세 설명, 스크린샷, 버전 정보 등을 확인할 수 있습니다.
  6. MCP 선택 완료: 원하는 MCP를 찾고 상세 정보까지 확인했다면 다음 단계로 진행할 준비가 완료됩니다.
3

MCP 설치 진행

찾은 MCP를 클로드 앱에 설치해보세요.

  1. 설치 버튼 클릭: 상세 정보 페이지에서 설치 또는 다운로드 및 설치 버튼을 클릭합니다.
  2. 자동 다운로드 및 설치: mcplink 데스크탑 앱이 자동으로 해당 MCP 파일을 다운로드하고, 사용자의 클로드 앱 설치 경로를 찾아 올바른 위치에 파일을 설치합니다.
  3. 관리자 권한 허용: 설치 과정 중 컴퓨터 시스템에서 관리자 권한 허용 등을 요청할 수 있습니다. 예 또는 허용을 클릭하여 진행합니다.
  4. 설치 완료 메시지 확인: 설치가 완료되면 앱 화면에 설치 완료 메시지가 표시됩니다.
  5. 다음 단계로 이동: 설치가 완료되었다면, 이제 클로드 앱을 재시작할 준비가 되었습니다. 다음 단계로 이동하세요.
알림

설치 과정이 자동으로 진행되지만, 진행 상황을 주의 깊게 확인해주세요.

4

클로드 앱 재시작 및 MCP 적용 확인

설치한 MCP를 클로드 앱에 적용하려면, 앱을 재시작해야 합니다. 재시작 후 MCP가 제대로 적용되었는지 확인하세요.

안내
  1. 클로드 앱 종료: 현재 실행 중인 클로드 앱을 완전히 종료합니다. (클로드 앱 좌측 상단 설정 - 종료 메뉴 이용)
  2. 클로드 앱 재실행: 클로드 앱을 다시 실행합니다.
  3. MCP 적용 확인: 클로드 앱이 재시작되면, 설치한 MCP가 정상적으로 적용되었는지 확인합니다.
  4. 추가 확인 사항: 만약 MCP가 정상적으로 적용되지 않았다면, 설치 과정 중 오류가 발생했거나 클로드 앱과의 호환성 문제가 있을 수 있습니다. 이 경우, 이전 단계를 다시 확인하거나 팀 SCP-201에 문의해주세요.
5

설치된 MCP 확인 (선택 사항)

클로드 앱을 재시작한 후, 앱 내에서 설치한 MCP가 정상적으로 적용되었는지 확인해볼 수 있습니다.

문제 해결 및 문의

mcplink 사용 중 문제가 발생하거나 도움이 필요하시면 언제든 문의해주세요. mcplink 웹사이트의 FAQ 페이지 를 확인하거나, 팀 SCP-201에 직접 문의하여 상세한 도움을 받으실 수 있습니다.
